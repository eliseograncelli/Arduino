Sensor de proximidad.
Si no hay objeto alguno en el rango detectado por el sensor no hace nada.
Si hay un objeto en el rango detectable por el sensor pero lejos (entre 150 y 300cm) prende el led verde constantemente. 
Si el objeto se encuentra a media distancia del sensor (entre 50 y 150cm) prende y apaga el led amarillo y tambien emite un sonido el buzzer.
Si el objeto esta muy cerca (entre 50 y 0cm) del sensor prende y apaga el led rojo, con mas frecuencia que el amarillo, tambien alerta el buzzer.
